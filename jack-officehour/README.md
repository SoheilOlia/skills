# jack-officehour

Evaluates any work artifact against Block's intelligence-native operating model (Block 2.0). Scores across 7 dimensions, catches alignment theater, identifies gaps, and shows exactly how to make the work A+.

## Install

### Amp

```bash
curl -sL https://raw.githubusercontent.com/SoheilOlia/skills/main/install.sh | bash -s jack-officehour
```

### Claude Code

```bash
curl -sL https://raw.githubusercontent.com/SoheilOlia/skills/main/jack-officehour/claude-code/jack-officehour.md -o .claude/commands/jack-officehour.md
```

Then use `/jack-officehour` in any Claude Code session.

### Claude (web)

Copy the contents of [`SKILL.md`](SKILL.md) (everything below the `---` frontmatter) into your **Project Instructions** or **Custom Instructions**.

## Usage

Share any work artifact and say `/jack-officehour` or:

- "Pressure-test this against Block 2.0"
- "Grade this against what Jack is pushing Block toward"
- "Review this like I'm about to discuss it in office hours with the CEO"
- "Is this strategically aligned with where Block is going?"

## What It Scores

| Dimension | What it measures |
|-----------|-----------------|
| Capability leverage | Does this create a reusable primitive? |
| World-model value | Does this create structured signal or durable insight? |
| System-learning value | Does this teach the machine? |
| Coordination reduction | Does this reduce handoffs and ambiguity? |
| Intelligence-layer contribution | Does this improve decision quality? |
| Compounding value | Will future work build on this? |
| Human judgment design | Is automation vs. human judgment clearly separated? |
