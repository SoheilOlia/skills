# jack-officehour

Block 2.0 strategic review skill for [Amp](https://ampcode.com).

Evaluates any work artifact against Block's intelligence-native operating model. Scores across 7 dimensions, catches alignment theater, identifies gaps, and shows exactly how to make the work A+.

## Install

```bash
curl -sL https://raw.githubusercontent.com/SoheilOlia/skills/main/install.sh | bash -s jack-officehour
```

## Usage

Share any work artifact and say `/jack-officehour` or:

- "Pressure-test this against Block 2.0"
- "Grade this against what Jack is pushing Block toward"
- "Review this like I'm about to discuss it in office hours with the CEO"
- "Is this strategically aligned with where Block is going?"
